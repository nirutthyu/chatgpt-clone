# ChatGPT UI Clone 

This is the clone of ChatGPT UI made using **React** and **Material UI** 

## Theme Toggle Feature
The application supports **Light Mode** and **Dark Mode** 

- `createTheme()` dynamically generates a theme object based on the current `mode`.
- The application is wrapped inside MUI‚Äôs `ThemeProvider` and the generated theme is passed as a prop.
- `ThemeProvider` internally uses React Context to make the theme available to all components.
